# DDoS-Alerts
Checks If Your Server Is Being DDoSed And Sends A Notification To A Discord Webhook With How Much MB Is Being Hit. 



# Installation


apt install python

apt install python-pip

pip install requests

apt install tcpdump

chmod +x log.sh

./log.sh
